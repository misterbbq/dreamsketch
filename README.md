#  Love Space

Site web collaboratif hébergé sur GitHub Pages.

## Fonctionnalités
- Accès par mot de passe journalier
- Agenda partagé en temps réel
- Multi-utilisateur anonyme
- Optimisé mobile

## Installation
1. Créer un projet Firebase
2. Remplir firebase.js
3. Push sur GitHub
4. Activer GitHub Pages

⚠️ Mot de passe côté client (usage non sensible)
